# TikTok Live Invite App - User Guide

## How to Use
1. Visit app URL.
2. Click "Login with TikTok".
3. Enter your live stream username/ID.
4. Add up to 5 invitee usernames.
5. Click "Compose Invite" to preview post.
6. Click "Share" to open TikTok and post (manual) or enable automation for n8n triggers.
7. View dashboard for history/analytics.

## Tips
- Ensure you're live before sharing.
- Posts are public; tag wisely.
- For best reach, share when active.
